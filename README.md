## Development in containers
### prerequisites:
- Docker
- yarn
- sops

### usage:
1. ```git clone```
2. ```yarn decrypt```
3. ```yarn dev:container```
4. ```yarn migrate:up```


## Development locally

### prerequisites:
- Docker
- yarn 
- nodemon
- sops

### usage:
1. ```git clone```
2. ```yarn decrypt```
3. ```yarn install```
4. ```yarn dev```
5. ```yarn migrate:up```


## Start a production env app
1. ```git clone```
2. ```yarn decrypt```
3. ```yarn container:prod```
